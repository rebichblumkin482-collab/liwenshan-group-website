# 李文山副教授课题组展示网站 — 设计文档

## 项目概述

为上海交通大学机械与动力工程学院李文山副教授课题组（微纳工程科学全国重点实验室）构建一个交互丰富、体验现代的学术展示网站，面向学界同行、潜在学生及公众展示课题组研究方向、研究成果与招生信息。

参考风格：北京大学徐克课题组主页（博客式动态内容 + 卡片交互布局）。

## 技术栈

| 层面 | 选型 |
|------|------|
| 框架 | React 18 + TypeScript |
| 构建 | Vite 5 |
| 样式 | Tailwind CSS 3 |
| 路由 | react-router-dom (HashRouter) |
| 动画 | framer-motion（滚动入场、hover动效、页面过渡） |
| 图标 | lucide-react |
| 轮播 | Swiper.js |
| 部署 | 纯静态文件，GitHub Pages / 任意静态托管 |

## 页面结构与交互设计

网站使用 HashRouter 实现多页面导航（非单页滚动），每页独立路由。

### 路由结构

```
/              首页（Hero + 简介 + 研究方向总览 + 最新新闻摘要）
/research      研究方向详情页
/advisor       导师介绍页
/members       团队成员页
/publications  论文成果页
/news          新闻动态页（博客式）
/recruitment   招生信息页
```

---

### 1. Navbar（全局导航栏）

**视觉**：交大红 `#BE1E2D` 背景，固定顶部，z-index最高
**内容**：
- 左侧：课题组 Logo/名称 + SJTU标识
- 右侧：首页 / 研究方向 / 导师介绍 / 团队成员 / 论文成果 / 新闻动态 / 招生信息
- 当前页面菜单项：白色下划线 + 加粗高亮
**交互**：
- 页面切换时平滑过渡（framer-motion AnimatePresence）
- hover菜单项：白色半透明底色过渡
- 滚动时导航栏加阴影（shadow-md）
- 移动端：汉堡菜单展开/收起动画，全屏覆盖菜单面板从右侧滑入

---

### 2. 首页（HomePage）

#### 2a. Hero 头图区
- 全屏高度（100vh）
- 背景：CSS渐变（交大红→深蓝）+ 粒子/几何图形动态背景（纯CSS实现）
- 内容居中：课题组名称大字 + 英文副标题 + 所属学院/实验室
- **交互**：
  - 标题文字逐字淡入动画（framer-motion stagger）
  - 向下滚动指示箭头：上下浮动的bounce动画
  - 背景粒子缓慢移动（CSS animation）

#### 2b. 课题组简介（首页摘要）
- 左文右图布局
- **交互**：
  - 滚动进入视口时从左侧滑入（framer-motion useInView）
  - 右侧图片微缩放hover效果

#### 2c. 研究方向卡片（首页摘要）
- 4张卡片水平排列，每张带对应图标（lucide-react）
- **交互**：
  - 卡片hover：上浮平移（translateY -8px）+ 阴影加深 + 图标微放大
  - 卡片click：跳转到 `/research` 详情页
  - 滚动进入时卡片依次从下方滑入（stagger 0.1s）
  - 每张卡片有独特的强调色条（左上角小色块，取自交大配色体系）

#### 2d. 最新新闻（首页摘要）
- 显示最近3条新闻的卡片列表
- 每条：日期标签 + 标题 + 摘要截断
- **交互**：
  - hover：左侧红色边框出现动画
  - click：跳转 `/news` 页并定位到该条新闻
  - "查看全部 →" 链接到新闻页

---

### 3. 研究方向页（ResearchPage）

- 顶部大标题 + 简短引言
- 4个方向各自展开成大卡片，包含：
  - 图标 + 方向名称
  - 详细描述（2-3句话）
  - 关键词标签（Tag形式）
  - 代表性图片/示意图
- **交互**：
  - 卡片hover：图片区域放大裁剪效果
  - 关键词标签hover：变为主色填充
  - 卡片进入视口：淡入+上移
  - 可展开/收起详细信息（accordion效果）

---

### 4. 导师介绍页（AdvisorPage）

- 上部：大照片（圆角矩形）+ 姓名、职称、邮箱（可点击复制）
- 中部：教育/工作经历 **交互式时间线**
  - 垂直时间线，左右交替排列
  - 每个节点：年份 + 机构名称 + 学位/职位 + 简述
  - **交互**：滚动到时间线区域时，节点依次从两侧滑入；hover节点放大+发光效果
- 下部：代表性成果卡片（3-4张）
  - **交互**：hover翻转显示详细信息（CSS 3D transform）
- 联系方式：一键复制邮箱按钮

---

### 5. 团队成员页（MembersPage）

- 顶部：分类筛选标签 — 全部 / 教师 / 博士生 / 硕士生 /  alumni
- 卡片网格布局（桌面4列→平板2列→手机1列）
- 每张卡片：
  - 正面：照片 + 姓名 + 身份 + 入学年份
  - hover：照片灰度→彩色过渡 + 卡片上浮
- **交互**：
  - **点击卡片弹出Modal**：显示详细信息（教育背景、研究方向、邮箱、个人简介，如有）
  - Modal：居中弹出，背景半透明遮罩，内容区滚动；关闭按钮 + 点击遮罩关闭 + Esc关闭
  - 筛选标签切换时卡片有 layout 动画（framer-motion layoutId）
  - 照片懒加载

---

### 6. 论文成果页（PublicationsPage）

- **顶部工具栏**：
  - 年份筛选按钮组（All / 2024 / 2023 / 2022 / ...）
  - 搜索框：实时过滤论文标题/作者/期刊（输入即过滤，300ms debounce）
- 论文列表：
  - 每条论文：编号 + 作者列表 + **标题（加粗）** + 期刊名斜体 + 卷期页码 + 年份标签
  - 代表性论文左侧有红色竖线标记
- **交互**：
  - 年份筛选切换时，论文列表有淡入过渡
  - 搜索过滤实时更新，无结果时显示空状态提示
  - 论文条目hover：浅灰背景
  - DOI链接（如有）：外链图标，新标签页打开
  - 点击论文行 → 展开摘要/abstract（accordion）

---

### 7. 新闻动态页（NewsPage）— 博客式

- **左侧主内容区（2/3宽度）**：
  - 新闻列表，每条为一张卡片：
    - 封面图（可选）+ 发布日期 + 标题 + 摘要
    - 底部：分类标签（如：学术会议 / 论文发表 / 组内活动 / 获奖）
  - **分页**：底部分页组件，页码按钮，当前页高亮
- **右侧边栏（1/3宽度，移动端隐藏）**：
  - **分类标签云**：点击筛选对应分类的新闻
  - **时间归档**：按年月折叠列表，点击展开/收起某月
- **交互**：
  - 点击新闻卡片 → 进入详情页 `/news/:id`
  - 详情页：大标题 + 发布日期 + 正文内容（支持富文本）+ 底部"返回新闻列表"
  - 分类标签点击 → 筛选动画
  - 页面切换：淡入过渡

---

### 8. 招生信息页（RecruitmentPage）

- 招生方向卡片（2-3张）
- 申请要求列表（图标 + 文字）
- 课题组优势亮点（数据统计数字动画：如已发表论文数、在研项目数等，数字滚动递增效果）
- **交互**：
  - 数字统计：进入视口时从0滚动到目标数字（count-up动画）
  - "联系导师"按钮：hover放大+发光阴影
  - 点击按钮弹出联系表单Modal（可选实现），或直接 mailto:

---

### 9. Footer（全局页脚）

- 深蓝 `#1A2B4F` 背景
- 三列布局：联系方式 / 快速链接 / 学校信息
- 底部版权条
- **交互**：
  - 邮箱点击复制到剪贴板 + toast提示"已复制"
  - 快速链接hover：下划线动画

---

### 10. 全局交互元素

| 元素 | 描述 |
|------|------|
| 回到顶部按钮 | 滚动超过一屏时出现，圆形按钮，hover放大，点击平滑滚动到顶部 |
| 页面过渡动画 | 路由切换时使用 AnimatePresence 做淡入+微移位过渡 |
| 滚动进度条 | 页面顶部极细的红色进度条，指示当前页面阅读进度 |
| 图片懒加载 | 所有图片使用 loading="lazy" + IntersectionObserver |
| Toast提示 | 复制邮箱等操作后的轻提示，右上角弹出，3秒自动消失 |
| 404页面 | 趣味404页面，带返回首页按钮 |

## 数据模型

所有内容数据存放在 `src/data/` 目录下，通过 TypeScript 模块导出：

```
src/data/
├── advisor.ts       # 导师信息（姓名、照片、经历时间线、成果）
├── members.ts       # 团队成员列表（姓名、照片、身份、方向、简介）
├── publications.ts  # 论文列表（标题、作者、期刊、年份、DOI、摘要）
├── news.ts          # 新闻动态（标题、日期、分类、摘要、正文、封面图）
├── research.ts      # 研究方向（名称、图标、描述、标签、图片）
├── recruitment.ts   # 招生信息（方向、要求、统计数据）
└── site.ts          # 站点全局配置（名称、联系方式、导航菜单等）
```

修改内容只需编辑对应数据文件，无需改动组件代码。

## 响应式断点

| 断点 | 布局 |
|------|------|
| ≥1024px | 桌面端：多列网格、完整导航、侧边栏可见 |
| 768–1023px | 平板端：2列网格、紧凑间距、侧边栏隐藏 |
| <768px | 移动端：单列堆叠、汉堡菜单、Modal全屏 |

## 部署方案

1. `npm run build` → 产出 `dist/` 静态文件
2. 部署到 GitHub Pages：开启仓库 Pages 功能，指向 `dist/` 或使用 `gh-pages` 分支
3. 如需自定义域名，添加 CNAME 文件并配置 DNS
